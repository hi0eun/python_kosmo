# 파이썬 공부방식
    A -> B -> C -> D
    순서대로 공부하다가 A가 기억이 나지않는다면 다시 A로 돌아가서 공부해라

    `정리(복습)

# 파이썬 기본 규칙
    - 들여쓰기는 4 스페이스(탭)
    - 한 줄은 최대 80자 이내로
    - 불필요한 공백은 피함

# PEP8 규칙 ( 가능하면 지키는 코딩 방식 )
    - = 연산자는 1칸 이상 띄우지 않는다 ( 요즘 지켜지지 않음 )
            a = 12  ->  a=12
    - 변수명에 소문자 l(엘), 대문자 O(오), 대문자 I (아이)는 금한다
            lIOO = 100  (엘아이오오)
    - 주석은 항상 갱신하고, 불필요한 주석을 지운다
    - 함수명은 소문자로 구성하고 필요하면 밑줄로 나눈다

# 파이썬은 파이써닉하게 코딩하자!


#[파이썬의 특징]

1.간단하고 배우기 쉽다 ( 읽기 쉬운 언어 )
2.인터프리터 언어( 컴파일언어가 아님) <-> 대표적인 컴파일 언어 : c / java
-인터프리터는 동시통역 컴파일은 통번역
-자바스크립트와 유사
3.강력한 라이브러리
4.범용적 언어 (R(데이터분석언어)에 비교해서)
    ex) pandas 라이브러리 vs. R

#[파이썬의 약점]
1.하드웨어를 제어하거나 연산,반복 프로그래밍은 부적합
2.GIL ( Global INterpreter Lock) : 쓰레드 사용이 수월하지 않음
